# 🍾 Truth or Dare

The Ultimate Party Game! Play **Truth or Dare** offline with your friends on the same device, or host an online room to play together on different phones! Built with ❤️ using [Streamlit](https://streamlit.io/).

## 🎮 Features
- **Offline Mode:** Add your friends' names and pass the phone around.
- **Online Multiplayer Mode:** Create a room code and let your friends join from their own devices! The bottle spins in real-time for everyone!
- **Categories:** Choose from Popular, Xtreme, Couple, Spicy, Kids, and Office categories.
- **Spin the Bottle:** Cool spinning animation and sound effects.
- **Live Chat:** Chat with your friends directly in the game lobby.

## 🚀 How to Play Online
1. Host a game and create a Room Code.
2. Share the code with your friends.
3. Friends click **Join Game** and enter the code.
4. The Host starts the game!

## 💻 How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/Vishvadip11/Truth-and-Dare.git
   cd Truth-and-Dare
   ```
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the app:
   ```bash
   streamlit run app.py
   ```

Enjoy the game! 🎉
